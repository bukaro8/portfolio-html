E-Commerce Platform

This project is a fully responsive e-commerce platform built with Next.js, leveraging its powerful App Router and Server Actions for seamless performance and dynamic routing. The frontend is styled using Tailwind CSS, ensuring a modern and sleek design.

For data management, the platform integrates Prisma with a PostgreSQL database, offering a robust and scalable backend. User authentication is handled through NextAuth.js, enabling secure login via OAuth providers.

Key Features:

  • Stripe integration for secure payments
  • Dynamic product pages with server-side rendering (SSR)
  • State management using Zustand
  • Optimised images with Next.js Image component
